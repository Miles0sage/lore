# ReAct Agent — Reasoning + Acting Loop

Demonstrates the **ReAct** (Reason → Act → Observe) pattern using pure Python stdlib
and two fake tools: `calculator` and `search`.

## What it shows

The agent solves a multi-step word problem:
> "What is the combined population of France and Germany, and what percentage does France represent?"

At each step the agent:
1. **Thinks** — produces a natural-language reasoning trace
2. **Acts** — selects a tool and calls it with specific input
3. **Observes** — receives the tool result and incorporates it into the next thought

## Run

```bash
python3 examples/react_agent/main.py
```

## Key classes

- `ReActAgent` — executes the Thought / Action / Observation loop
- `tool_calculator` — safely evaluates arithmetic via `eval` with restricted builtins
- `tool_search` — returns canned results from an in-memory knowledge base
- `Step` — dataclass recording each loop iteration for the full trace

## Design note

In production, each `thought` + `action` is generated by an LLM call. The scripted
trace here makes the pattern visible without requiring an API key.
