60-Second Demo

Four commands. Four controls. Any framework.

Each scaffold generates production-ready Python. Drop it into your existing project — LangGraph, CrewAI, AutoGen, or plain Python.

⏱️
Cost Guard
Hard stop before budget burns
$ lore scaffold cost_guard
class CostGuard:
  # hard stop at budget_tokens
  # warns at 80% threshold
  # raises CostGuardExceeded
  # zero external dependencies

guard = CostGuard(budget_tokens=100_000)
guard.consume("llm_call", tokens=2400)
🔴
Circuit Breaker
Stop cascades before they drain budget
$ lore scaffold circuit_breaker
CLOSED → (5 failures) → OPEN
OPEN → (30s wait) → HALF_OPEN
HALF_OPEN → (1 success) → CLOSED

breaker = CircuitBreaker(
  name="openai_api",
  failure_threshold=5
)
result = await breaker.call(fn)

Getting Started — New to this?

Five steps from zero to production-hardened.

Never used a reliability harness before? Start here. Each step takes under 2 minutes.

1

Install

Pure Python. No mandatory dependencies. Works on Python 3.10+. Zero config.

pip install lore-agents
2

Audit your existing codebase

Find out what's missing before you ship. Uses Gemini 2.5 Pro to check 4 reliability dimensions.

lore audit .
# → JSON report: what's missing, severity, exact fix command
# → run it on your existing code, no changes required
3

Scaffold what's missing

One command generates the code. Write it to a file and wire it in.

lore scaffold cost_guard -o src/cost_guard.py
lore scaffold circuit_breaker -o src/circuit_breaker.py
lore scaffold dead_letter_queue -o src/dlq.py
# list all 19 patterns:
lore scaffold --list
4

Search the knowledge base

76 production articles explain every pattern — when to use it, when it fails, real examples.

lore search "cost guard"
lore search "retry failure handling"
lore read circuit-breaker-pattern-for-ai-agents
lore list
# full-text BM25 search, zero API calls, works offline
5

Teach your AI assistant before it writes code

One command injects LORE's 15 pattern rules into your Claude Code or Cursor project. Your assistant learns the patterns before it writes a line.

lore install /path/to/your/project

Installed:
  .claude/CLAUDE.md          ← 15 pattern rules
  .claude/hooks/pre_tool.py  ← blocks anti-patterns before written
  .claude/skills/lore.yaml   ← scaffold shortcuts as slash commands

Claude now knows:
  ✓ Never write retry without circuit breaker
  ✓ Always add cost guard before LLM calls
  ✓ Use cheapest model tier that fits the task
