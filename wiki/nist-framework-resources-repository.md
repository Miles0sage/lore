# NIST Framework Resources Repository

## Overview
The `NIST-framework-resources` repository, maintained by Craig Erickson (`craigericksondpo`), hosts technical resources, documentation, and crosswalks aligned with the NIST Privacy Framework, Cybersecurity Framework, and AI Risk Management Framework. The repository functions as a public staging environment for materials intended for official NIST distribution.

## Primary Contribution
The initial artifact published to the repository is `CCPA-Regs-to-SP800-53r5-to-pf-mappings.xlsx`. This spreadsheet establishes a three-way regulatory and control mapping between:
- The CCPA Final Regulations, effective 3/29/2023.
- A selected subset of SP 800-53 Revision 5 controls that support legal compliance with the CCPA Final Regulations.
- Subcategories within the Privacy Framework Core (designated as "Privacy Framework Outcomes").

## Repository Workflow & Maintenance
Contributions accepted for inclusion into official NIST resource repositories will be maintained by NIST by the author, Craig Erickson. The repository operates as a staging layer where external users can submit:
- Issues and comments
- Pull requests
- General updates and documentation artifacts

Once validated, staged updates are propagated to the corresponding NIST repositories. All supporting documentation and version history are retained within this GitHub repository. The latest commit (`7f752d3`) was recorded on Dec 8, 2023.
