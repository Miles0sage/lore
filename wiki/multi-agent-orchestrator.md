## Project Architecture
The repository is structured into the following directories and configuration files:
- `chapters/` - Implementation files mapped to each of the 15 instructional chapters.
- `scripts/` - Automation utilities, including the `run-all-demos` execution script.
- `shared/` - Cross-module utility libraries and helper functions.
- `src/` - Core production modules for the orchestrator runtime.
- `.env.example` - Environment variable configuration template.
- `.gitignore` - Version control exclusion patterns.
- `CLAUDE.MD` - AI assistant context and configuration directives.
- `Dockerfile` - Containerization specification for deployment.
